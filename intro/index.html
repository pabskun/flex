Formato visual

Es la forma mediante la cuál los elementos se acomodan dentro de la página.

Existen dos formas principales en la que los elementos se pueden acomodar:

Elementos en línea
Son aquellos que se acomodan uno a la par del otro. Por ejemplo los elementos de enlaces, input, label, span.
Elementos en bloque
Son aquellos elementos que se acomodan por defecto uno abajo del otro, y ocupan el 100% del ancho de la página

Adicionalmente existen otras posibilidades de modificar el formato visual con los valores inline-bloc, list, table, entre otros.

Flexbox

El el formato visual que se ha incluido más recientemente y permite crear páginas web que sean más fluidas, y además provee múltiples facilidades para acomodar los elementos dentro de una página.
